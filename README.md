# Problem Solving
| Name          | Worst Case    | Average Case  | Best Case     |
| ------------- |:-------------:|:-------------:|:-------------:|
| Linear Search | O(n)          | O(n)          | O(1)          |
| Binary Search | O(log2n)      | O(log2n)      | O(1)          |
| Insertion Sort| O(n^2)        | O(n^2)        | O(n)          |
| Selection Sort| O(n^2)        | O(n^2)        | O(n^2)        |
| Bubble Sort   | O(n^2)        | O(n^2)        | O(n)          |
